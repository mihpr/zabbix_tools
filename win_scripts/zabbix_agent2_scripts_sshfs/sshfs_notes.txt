How to access a Agent2 and plugin sources located in a git repo on Linux guest from Windows host:


https://winfsp.dev/rel/
Download WinFsp Installer (and install)
Download SSHFS-Win (x64) (and install)

To solve problems with permissions, do the following:

Open registry editor.
https://github.com/winfsp/sshfs-win/issues/68#issuecomment-606845664

Set the following registry key:
HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\WinFsp\Services\sshfs registry key (in Win64) 
value CommandLine to:
svc %1 %2 %U -ovolname=lab-u22 -oidmap=user -ouid=-1 -ogid=-1 -oumask=000 -ocreate_umask=000 -omax_readahead=1GB -oallow_other -olarge_read -okernel_cache -ofollow_symlinks

Right click "This PC" on Windows host.
Add new network location, which is placed at the guest Linux machine. Provide SFTP (SSH) access.

For example:
\\sshfs\mikhail@192.168.56.201

-ovolname=lab-u22 -odebug -ologlevel=debug1 -oStrictHostKeyChecking=no -oUserKnownHostsFile=/dev/null -oidmap=user -ouid=-1 -ogid=-1 -oumask=000 -ocreate_umask=000 -omax_readahead=1GB -oallow_other -olarge_read -okernel_cache -ofollow_symlinks -oPreferredAuthentications=password -opassword_stdin
